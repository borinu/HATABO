カルーセルプラグイン

機能

  • レスポンシブ対応
  • フリック対応
  • インジケータ対応
  • ループ/往復切り替え
  • 複数枚の同時移動対応

使用方法

サンプルで使用しているクラスのうち、先頭が「jsc」で始まるクラスを適切に付与する。

左右移動のトリガーとインジケータは任意。

インジケータを使用する場合、全てのインジケータを静的に記述しておく方法と、1つだけ記述しておいたインジケータを、自動的に複製する方法を選択することが可能。

オプション

slideAuto
真偽値(デフォルト値 : true)
true : 一定間隔で自動的に移動する, false : 自動的に移動しない
slideInterval
数値(デフォルト値 : 5000)
カルーセルの移動間隔(msec)。slideAutoがtrueの時のみ有効。
slideTime
数値(デフォルト値 : 300)
カルーセルの移動時間(msec)
slideLoop
真偽値(デフォルト値 : true)
true : ループする, false : ループしない
flickEnabled
真偽値(デフォルト値 : true)
true : フリック可能, false : フリック不可
flickThreshold
数値(デフォルト値 : 75)
フリック判定の距離(px)。短いほどフリックの反応が良くなる。flickEnabledがtrueの時のみ有効。
indicatorAdjustment
真偽値(デフォルト値 : false)
true : 1つのインジケータを複製する, false : インジケータを複製しない。
imageCountPerView
数値(デフォルト値 : 1)
1画面に表示される画像の枚数。
destroy
真偽値(デフォルト値 : false)
true : カルーセル制御用オブジェクトを削除する(他のオプションは無視される), false : カルーセル制御用オブジェクトを削除しない
